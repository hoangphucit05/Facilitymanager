# Static assets

- `css/` — global, component, and page styles
- `js/` — application scripts (`api/`, `auth/`, `components/`, `pages/`, `shared/`, `student/`)
- `images/` — logos, avatars, illustrations
- `icons/` — SVG icons (kebab-case names)
- `fonts/` — local fonts (if any)

Locale JSON files are in `src/locales/` (sibling of `assets/`).
