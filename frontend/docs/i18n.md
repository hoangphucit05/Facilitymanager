# i18n — Đa ngôn ngữ

Hỗ trợ **vi** (mặc định), **en**, **ja** — client-side, không phụ thuộc backend.

## File bundle

| File | Ngôn ngữ |
|------|----------|
| `src/locales/vi.json` | Tiếng Việt |
| `src/locales/en.json` | English |
| `src/locales/ja.json` | 日本語 (có `<ruby>` furigana) |

Sau build:

- `dist/locales/{lang}.json`
- `dist/i18n/{lang}.json` (alias — build script copy locales → i18n)

## Module chính

| File | Global |
|------|--------|
| `shared/i18n-manager.js` | `window.FmI18n` |
| `shared/lang-switcher.js` | UI chọn ngôn ngữ + furigana JA |

## Khởi tạo

Hầu hết trang dashboard:

```html
<script src="../../assets/js/shared/i18n-manager.js"></script>
<script src="../../assets/js/shared/lang-switcher.js"></script>
```

Cuối body hoặc sau DOM ready:

```javascript
window.FmI18n.init();
```

Event sau khi áp dụng xong: `fm-i18n-applied` — sidebar / menu-links lắng nghe để cập nhật lại nhãn.

## Lưu locale

| Key localStorage | Giá trị |
|------------------|---------|
| `fm_ui_lang` | `vi` \| `en` \| `ja` |
| `fm_ui_furigana` | `1` \| `0` (chỉ JA) |

`lang-switcher.js` ghi key và dispatch `fm-lang-change`.

## Markup HTML

| Attribute | Tác dụng |
|-----------|----------|
| `data-i18n="key.path"` | Text node (`textContent`) |
| `data-i18n-html="key.path"` | HTML tin cậy từ bundle |
| `data-i18n-placeholder="key"` | `placeholder` input |
| `data-i18n-title="key"` | `title` tooltip |
| `data-i18n-aria-label="key"` | ARIA |
| `data-i18n-alt="key"` | `alt` ảnh |
| `data-i18n-params='{"n":1}'` | Tham số `{{n}}` trong chuỗi |

Ví dụ:

```html
<h1 data-i18n="assets.pageTitle">Quản lý tài sản</h1>
<input data-i18n-placeholder="userForm.phFullName" />
<th data-i18n="departments.colRoomCode">Mã phòng</th>
```

Fallback: nếu không có bundle, giữ text tĩnh trong HTML.

## API JavaScript

```javascript
// Dịch một key
FmI18n.t("menu.assets");
FmI18n.t("categories.confirmDelete", { name: "Ghế" });

// Áp dụng toàn trang hoặc subtree
FmI18n.apply(document.querySelector("main.content"));

// Đổi ngôn ngữ
await FmI18n.setLocale("en");

// Locale hiện tại
FmI18n.getLocale(); // "vi"
```

## Cấu trúc key JSON (gợi ý)

```json
{
  "nav": { "language": "Ngôn ngữ", "logout": "Đăng xuất" },
  "menu": { "assets": "Tài sản", "users": "Người dùng" },
  "departments": { "colRoomCode": "Mã phòng", "loading": "Đang tải…" },
  "assets": { "pageTitle": "Quản lý tài sản" },
  "categories": {
    "views": {
      "default": { "title": "...", "columns": ["...", "..."] },
      "nguon_kinh_phi": { "title": "..." }
    }
  }
}
```

Key view danh mục: `categories.views.{viewKey_with_underscores}.*` — `main.js` map hash `#nguon-kinh-phi` → `nguon_kinh_phi`.

## Sidebar i18n

`sidebar.js` — `MENU_TITLE_TO_I18N_KEY`: map title tiếng Việt/Anh/Nhật từ API → key `menu.*`.

Sau khi render menu: gọi `FmI18n.apply()` trên sidebar.

Menu tòa dùng `data-i18n-params='{"name":"E1"}'` + key `menu.buildingName`.

## Cache busting

`i18n-manager.js` — hằng `BUNDLE_REV` (vd. `"20260529f"`).

URL fetch: `locales/vi.json?v=20260529f` — **tăng `BUNDLE_REV`** khi thêm/sửa key JSON để tránh cache cũ.

## Furigana (tiếng Nhật)

Nếu chuỗi dịch chứa `<ruby>`, `FmI18n` gán `innerHTML` thay vì `textContent`.

Toggle furigana: checkbox trong lang-switcher → `fm_ui_furigana`.

## Trang không dùng lang-switcher

Một số trang tối giản (`room-detail.html`) có thể không load đủ stack i18n — text cứng tiếng Việt.

## Thêm ngôn ngữ mới

1. Thêm mã vào `SUPPORTED` trong `i18n-manager.js`
2. Tạo `src/locales/xx.json`
3. Cập nhật `lang-switcher.js` (option UI)
4. Tăng `BUNDLE_REV`
5. Bổ sung title map trong `sidebar.js` nếu menu API trả title mới

## Liên kết routing

Hash / query **không** đổi theo locale — chỉ nội dung text thay đổi. URL giữ nguyên, ví dụ:

- `categories.html#nguon-kinh-phi`
- `assets.html#danh-gia-tai-san`

`menu-links.js` match hash theo **text menu** (cả 3 ngôn ngữ) trong `categoryMenuHashMap` / `assetsMenuHashMap`.
